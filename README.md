# Car Contact System — Car Dashboard Theme

A multi-car contact verification system with QR codes and a dashboard-style UI.  
Users can scan a car’s QR code, enter the last 4 digits of the car number, and view owner information.  

---

## **Features**

- Multi-car support (Brezza & Dzire preloaded)  
- Car Dashboard UI theme (dark dashboard, neon highlights)  
- QR code generation for each car  
- Expiring encrypted tokens (24 hours)  
- Admin dashboard to view all cars  
- Rate limiting & IP protection  
- Fully free and ready for deployment
